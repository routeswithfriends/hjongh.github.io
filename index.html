My Mods

FBToggle

FBToggle is a mod that allows the user to toggle fullbright lighting with a hotkey, changeable in the Controls section. Also changes users' gamma back to the default value if the user has changed it previously. This was the first mod I ever released!

BetterBow

BetterBow draws custom hitboxes for all airborne arrows. This one was a pain in the ass to make, mainly because there was no good way for client-side verification of an arrow being airborne. Created so that I could improve my bow strafes.

CoordsDisplay

CoordsDisplay is your basic coordinate and fps display mod. X and Y coordinates are truncated to one decimal, while height is a whole number. Created so that I could view my coordinates in UHC without toggling f3.

SkywarsStats

SkywarsStats is a live session statistics display mod for Hypixel Skywars. It displays session kill, death, K/D, and number of games won and lost. It also has an FPS display. I haven't gotten around to releasing this mod yet.

LineRenderer

LineRenderer renders toggleable lines from the crosshair to visible players. Can be used to locate other players even if they are shifting or out of nametag-view zone. There is a HUD in the top-left corner displaying the current toggle setting.

StatusHUD

StatusHUD shows a player heads-up-display with the current status effects, their amplification, and duration. Only shows a handful of effects, such as Speed and Strength. Created to be as lightweight as possible.
